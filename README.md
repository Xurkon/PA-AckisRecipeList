## üîß Usage

### Basic Scanning

1. Open any tradeskill window (Alchemy, Blacksmithing, etc.)
2. Click the **Scan** button
3. Browse missing recipes with detailed acquisition info

### Slash Commands

- `/arl` or `/ackisrecipelist` - Open the main window
- **Shift + Click Scan** - Force rescan of current profession
- **Alt + Click Scan** - Clear all TomTom waypoints

### Finding Recipes

- **Left-click** a recipe to see detailed acquisition information
- **Right-click** a recipe to exclude it from future scans
- Use the **Filter** tabs to narrow down results
- Click **Acquire** names to set TomTom waypoints automatically
